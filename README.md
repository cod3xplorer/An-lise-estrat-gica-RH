📌 Visão Geral

Projeto desenvolvido em Power BI com foco em análise de RH, reunindo indicadores estratégicos para apoiar a tomada de decisão baseada em dados.

O dashboard permite acompanhar métricas relacionadas ao ciclo completo do colaborador, incluindo:

Contratações e desligamentos

Retenção

Turnover

Absenteísmo

Ausências

Horas extras

Massa salarial

Perfil dos colaboradores

A proposta é transformar dados operacionais em insights estratégicos para a área de RH.

🎯 Objetivos do Projeto
Monitorar indicadores críticos de RH

Identificar padrões de desligamento

Analisar retenção de colaboradores

Avaliar impactos financeiros do absenteísmo

Entender perfil demográfico da empresa

Auxiliar decisões de gestão de pessoas
